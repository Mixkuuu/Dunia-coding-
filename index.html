HTML

HTML digunakan untuk membuat kerangka website.

Apa Itu <div>?

<div> adalah wadah untuk mengelompokkan elemen. 

<div>
  <h1>Halo Dunia</h1>
  <p>Teks</p>
  <button>Klik</button>
</div>
